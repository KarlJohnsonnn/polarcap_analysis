# Paper draft figure gallery

- **PNGs / MP4s**: Latest paper-ready figures (fig01, fig12, fig13, spectral waterfall videos).
- **figure_captions.yaml**: Caption text for each file; edit to match the manuscript.
- **gallery.ipynb**: Jupyter notebook that loads all images/videos and displays them with captions. Run from repo root so `output/gallery` is found.

Figures are merged from `notebooks/output` and `output/gfx`; this directory holds the canonical set for the draft.
